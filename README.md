# TAX! — Driver App

Mobile web app for licensed taxi drivers (Casablanca pilot).

## What this is
- `index.html` — the entire driver app (map, live demand, pooling, landing radar)
- `vercel.json` — tells Vercel to serve it as a simple website

## How it goes live
This folder is uploaded to a GitHub repository, which Vercel publishes
automatically. Any change pushed to GitHub redeploys in ~30 seconds.
